# Environment Setup

## .vimrc

Create and edit the `~/.vimrc` file
```
touch ~/.vimrc
vim ~/.vimrc
```

Use
```
set number # set numbers
set tabstop=2 shiftwidth=2 expandtab # use 2 spaces instead of tab
set ai # autoindent: when go to new line keep same indentation
```
