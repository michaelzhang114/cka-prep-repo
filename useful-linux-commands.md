# Useful Linux Commands

### Copy, cut, paste

Yank the line `yy`
Delete the line `dd`
Paste on the line `p`


### Copy multiple lines

Select lines you want to copy in visual mode `V`
Then yank `y`
And paste `p`

### Cut by Field

Cut by delimiter ':', taking the first and third field
`cut test.txt -d ':' -f 1,3`
